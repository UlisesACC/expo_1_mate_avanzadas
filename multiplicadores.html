Ejemplo de Multiplicadores de Lagrange

Problema

Una caja rectangular sin tapa debe hacerse con $12 m^2$ de material. Determine el volumen máximo de esa caja.

Planteamiento

Sea $x$ la longitud, $y$ el ancho y $z$ la altura de la caja. Entonces el volumen de la caja es: $$V(x,y,z) = xyz$$ La restricción viene dada por el área de la caja: $$A(x,y,z) = xy + 2xz + 2yz = 12$$ Planteamos el problema de optimización con restricción: $$\text{Maximizar } V(x,y,z) = xyz$$ $$\text{sujeto a } g(x,y,z) = xy + 2xz + 2yz - 12 = 0$$

Método de Multiplicadores de Lagrange

Usando el método de multiplicadores de Lagrange se buscan los valores de $x, y, z$ y $\lambda$ tales que: $$\nabla V(x,y,z) = \lambda \nabla g(x,y,z) \quad \text{y} \quad g(x,y,z) = 0$$ Calculamos los gradientes: $$V_x = \lambda g_x, \quad V_y = \lambda g_y, \quad V_z = \lambda g_z, \quad g(x,y,z) = 0$$

Sistema de Ecuaciones

Lo que nos da el siguiente sistema de ecuaciones: $$yz = \lambda (y + 2z)$$ $$xz = \lambda (x + 2z)$$ $$xy = \lambda (2x + 2y)$$ $$xy + 2xz + 2yz = 12$$

Resolución

Resolviendo este sistema de ecuaciones, obtenemos: $$xyz = \lambda(2xz + xy)$$ $$xyz = \lambda(2yz + xy)$$ $$xyz = \lambda(2xz + 2yz)$$ Entonces $x = y = 2z$, obteniendo: $$4z^2 + 4z^2 + 4z^2 = 12$$ $$12z^2 = 12$$ $$z^2 = 1 \Rightarrow z = 1$$ Por lo tanto $x = y = 2$.

Resultado Final

Las dimensiones óptimas de la caja son: $$x = 2 \text{ m}, \quad y = 2 \text{ m}, \quad z = 1 \text{ m}$$ El volumen máximo es: $$V(2, 2, 1) = 4 \text{ m}^3$$
